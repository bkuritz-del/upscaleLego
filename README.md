# batch-print-upscale

Batch upscales images to a target physical size (e.g., 12in @ 300dpi => 3600x3600 px) using Pillow.
Designed for crisp, defined shapes (e.g., LEGO bricks).

## Setup
```bash
python -m venv .venv
# mac/linux
source .venv/bin/activate
# windows
# .venv\Scripts\activate

pip install -r requirements.txt
